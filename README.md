# xiaoUtil
## 安装

### maven安装

```xml
<dependency>
  <groupId>io.github.xiaoxunyao</groupId>
  <artifactId>java-utils-library</artifactId>
  <version>0.0.1-releases</version>
</dependency>
```

### Gradle安装

```gradle
implementation 'io.github.xiaoxunyao:java-utils-library:0.0.1-releases'
```
